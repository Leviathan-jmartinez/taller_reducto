# Inicio de flujos para informes unificados

## Informes Referenciales

### Informe de Articulos

Generar
El sistema muestra la vista Informes Referenciales.
El sistema muestra los tipos de informe disponibles segun los permisos del usuario.
El sistema carga las categorias disponibles para el filtro. Tabla consultada: `categorias`.
El usuario selecciona Articulos como tipo de informe referencial.
El sistema muestra u oculta los filtros correspondientes al tipo seleccionado.
El sistema muestra los filtros de estado, categoria, busqueda y cantidad de registros.
El usuario selecciona o ingresa los filtros deseados.
